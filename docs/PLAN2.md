### 1. RABBIT HOLE GENERATOR *(highest priority)*
**What:** User selects a starting topic → site generates a personalised sequence of 7–10 connected topics to go through, with a progress bar and "you've gone X% deep" indicator.
**Why:** Drives session time from avg 2 min → 20+ min. The #1 retention mechanism for this content.
**How:** JS-only, no backend needed. Hardcode the connection graph from the existing 422 topic slugs.

---

### 2. INTERACTIVE FLAT EARTH EXPLORER
**What:** Clickable flat earth map. User taps a location → sidebar opens with a topic stub or full page.
**250+ locations from the master list** covering every continent and the Antarctic ring.
**Why:** The most visual and shareable feature possible for this audience.
**How:** SVG or canvas map, prebuilt coordinate mapping JSON.

---

### 3. CONSPIRACY TIER LIST
**What:** Drag-and-drop tier list (S/A/B/C/D) where users rate how "real" each conspiracy is. Results can be saved/shared as an image.
**Why:** Extremely shareable, drives social traffic. TikTok content writes itself.
**How:** Tiersmaker-style UI, localStorage for saves, canvas export for sharing.

---

### 4. TIMELINE COLLAPSE MODE
**What:** A horizontal scrolling timeline showing key events across history, with Truth Archive topics pinned at relevant dates. "Official history" vs "what actually happened" dual-track.
**Why:** Context tool — shows patterns across decades. Very sticky.
**How:** Prebuilt JSON data + CSS horizontal scroll component.

---

### 5. SYMBOL DECODER
**What:** Visual reference library. User browses symbols (Masonic eye, Baphomet, obelisk, CERN logo, etc.) → click reveals the meaning + links to related topics.
**Why:** Accessible entry point for normies. Shareable. "I spotted this on the news" use case.
**How:** Image grid + modal system. Static content.

---

### 6. CLASSIFIED VAULT *(monetisation angle)*
**What:** A locked section of the site styled as a redacted government file. Requires email signup or paid membership to unlock.
**Content:** The most sensitive/advanced topics — adrenochrome, Dulce Base, 20 & Back, off-planet slave trade — all styled as "declassified files."
**Why:** Monetisation hook. Premium feel. Converts browsers into members.
**How:** Extend existing Stripe paywall system.

---

### 7. HIDDEN CONNECTIONS ENGINE
**What:** Select any two topics → site shows the connection path between them. E.g. "CIA → Bohemian Grove" → 3 hops. Like a conspiracy version of "6 Degrees of Kevin Bacon."
**Why:** Highly engaging, demonstrates depth, viral.
**How:** Prebuilt adjacency graph from existing cross-ref data.

---

### 8. MANDELA EFFECT MEMORY TEST
**What:** Quiz mode — show a question ("What was the Monopoly man's accessory?"), user answers before seeing the "real" answer. Tracks their score.
**Why:** Extremely shareable. "I scored 7/10 on the Mandela Effect test" posts itself.
**How:** Static JSON quiz data + scoring JS. No backend.

---

### 9. MYSTERY SOUND ARCHIVE
**What:** Curated collection of unexplained audio — The Bloop, The Hum, UVB-76, numbers stations, Hessdalen anomaly recordings, etc. Each with a player + topic page.
**Why:** Unique content type. Breaks up the text-heavy pages. Extremely creepy/shareable.
**How:** Hosted audio files (S3 or VPS) + custom player UI.

---

### 10. LIVE NEWS DECODER
**What:** An RSS aggregator pulling from mainstream news sources. For each story, a sidebar shows which Truth Archive topics it "connects to." User sees mainstream story → site links it to a deeper topic.
**Why:** Makes the site feel alive and current. Retention through recurring visits.
**How:** Serverless RSS fetch + keyword matching to topic slugs. Can run on the existing Lambda infrastructure.
